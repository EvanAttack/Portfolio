Poursuite d'études

HTML CSS PHP JavaScript Bootstrap PostegreSQL Apache

Contexte :

Dans le cadre de ma formation, on nous a demandé de concevoir une application web permettant de faciliter le choix de la poursuite d'études pour les étudiants. Cette plateforme offre également aux écoles et au responsable d'études la possibilité de consulter les candidatures, d’accéder aux notes des étudiants, de donner leur avis et de créer des notes agrégées.

Le projet a été développé en PHP, en suivant le modèle MVC pour structurer le code de manière modulaire et efficace. Nous avons utilisé Apache pour faire tourner le serveur, garantissant une exécution fluide de l'application.

Objectifs principaux :

Offrir aux étudiants une interface intuitive pour postuler dans les écoles de leur choix. Permettre aux écoles de consulter facilement les profils des candidats et agréger les données des notes. Fournir au responsable d’études des outils pour visualiser et agréger les données des notes. Importation de fichiers CSV qui permet de charger les données des étudiants de manière automatisée. Système d'authentification qui garantit un accès sécurisé pour les différents utilisateurs (étudiants, écoles, responsable d’études). Création automatique de fichiers PDF qui génère un rapport synthétique à partir des données. Affichage des résultats sous forme de graphiques interactifs pour une analyse plus intuitive.

Apprentissages

Ce projet m’a permis de : Approfondir mes connaissances sur la manipulation de fichiers CSV et la génération de PDF via PHP. Comprendre les principes de sécurisation des applications grâce à un système d'authentification. Expérimenter la visualisation de données avec des graphiques interactifs. Renforcer mes compétences en programmation structurée avec une architecture MVC.
